## 내용정리 ##
1. static으로 선언된 클래스 변수는 모든 객체에서 하나의 값을 바라본다.

2. static 블록은 생성자가 호출되기 전에 호출된다.

3. static 블록에서는 static변수, 메소드만 호출가능하다.

4. static 블록은 생성자가 불리지 않더라도, 클래스에 대한 참조가 발생하자마자 호출된다.

5. 기본자료형은 pass by value(원래값 바뀌지않음), 참조자료형은 pass by reference
(값 바뀜)

6. 타입...변수명 : 매개변수를 배열로 인식하여 값의 개수를 지정하지않고 넘겨줄 수 있다.
- printf(String format, Object... args)
